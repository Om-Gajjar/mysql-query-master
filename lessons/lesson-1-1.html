1.1 Introduction to SELECT

The cornerstone of retrieving data from a MySQL database is the SELECT statement. Its primary purpose is to query the database and return data that matches the specified criteria.

The most basic form of the SELECT statement retrieves all columns (using the wildcard *) from a specific table:


SELECT *
FROM table_name;

Selecting Specific Columns



Instead of retrieving all columns with *, you can specify the exact columns you want by listing their names after the SELECT keyword, separated by commas:




SELECT column1, column2
FROM table_name;




For example, to select only the name and email from the users table, you would use:




SELECT name, email
FROM users;
