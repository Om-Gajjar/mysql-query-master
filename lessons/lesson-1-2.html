1.2 Filtering Data with WHERE

Often, you don't want to retrieve *all* rows from a table. The WHERE clause allows you to specify conditions to filter the rows returned by your SELECT statement.

The basic syntax is:


SELECT column1, column2, ...
FROM table_name
WHERE condition;

The condition typically involves comparing a column's value using operators like:

  • = (Equal to)
  • != or <> (Not equal to)
  • > (Greater than)
  • < (Less than)
  • >= (Greater than or equal to)
  • <= (Less than or equal to)
  • LIKE (Pattern matching - often used with % wildcard)
  • IN (Checks if a value is within a list)
  • BETWEEN (Checks if a value is within a range)
  • IS NULL (Checks if a value is NULL)
  • IS NOT NULL (Checks if a value is not NULL)

You can combine multiple conditions using AND (both conditions must be true) and OR (at least one condition must be true).

For example, to find the user named 'Alice Wonderland':


SELECT id, name, email
FROM users
WHERE name = 'Alice Wonderland';

To find all users who signed up before 2024:


SELECT name, signup_date
FROM users
WHERE signup_date < '2024-01-01';
