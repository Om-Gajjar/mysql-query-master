1.3 Sorting Results with ORDER BY

By default, the order of rows returned by a SELECT query is not guaranteed. To sort the results based on one or more columns, use the ORDER BY clause.

The syntax is:


SELECT column1, column2, ...
FROM table_name
WHERE condition  -- Optional
ORDER BY column_to_sort_by [ASC | DESC], another_column [ASC | DESC], ...;
  • ASC (Ascending) is the default sorting order (A-Z, smallest to largest). You can omit it.
  • DESC (Descending) sorts in reverse order (Z-A, largest to smallest).

You can sort by multiple columns. The sorting is applied sequentially based on the order you list the columns.

Example: Get all users, sorted alphabetically by name:


SELECT name, email, signup_date
FROM users
ORDER BY name ASC; -- ASC is optional here

Example: Get all users, sorted by signup date (most recent first), and then by name alphabetically for users who signed up on the same date:


SELECT name, signup_date
FROM users
ORDER BY signup_date DESC, name ASC;
