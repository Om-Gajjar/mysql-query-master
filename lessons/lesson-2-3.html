2.3 Deleting Data (DELETE FROM)

The DELETE FROM statement is used to remove existing rows from a table.

Syntax:


DELETE FROM table_name
WHERE condition; -- EXTREMELY IMPORTANT!

WARNING: Just like UPDATE, the WHERE clause is critical. If you omit the WHERE clause, all rows in the table will be deleted!

The WHERE clause specifies which row(s) should be deleted, using the same conditions as SELECT and UPDATE.

Example: Delete the user with the email 'charlie@example.com':


DELETE FROM users
WHERE email = 'charlie@example.com';

Example: Delete all users who signed up before 2024:


DELETE FROM users
WHERE signup_date < '2024-01-01';

There is no way to undo a DELETE operation with standard SQL once it's committed (though database backups are used in real-world scenarios). Be very careful!
